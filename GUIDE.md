# Quick Start Guide

## Step 1: Get Your LinkedIn Profile PDF

1. Open LinkedIn and go to your profile
2. Click the **"More"** button (three dots)
3. Select **"Save to PDF"**
4. Your profile will download as a PDF file

## Step 2: Find a Job Description

1. Find a job posting you're interested in
2. Copy the entire job description including:
   - Job title
   - Required qualifications
   - Responsibilities
   - Preferred skills
   - Company information

## Step 3: Generate Your Resume

1. Open the app in your browser
2. Upload your LinkedIn PDF
3. Paste the job description
4. Click **"Generate ATS Resume"**
5. Wait 10-30 seconds for AI processing

## Step 4: Download and Use

1. Review the generated resume
2. Download as PDF or copy the text
3. Use the cover letter tab for a tailored cover letter
4. Submit your application!

## Tips for Best Results

✅ **DO:**
- Use a complete LinkedIn profile with detailed experience
- Paste the full job description (more detail = better matching)
- Review and customize the generated resume before sending
- Check the analysis tab for improvement suggestions

❌ **DON'T:**
- Upload incomplete or minimal LinkedIn profiles
- Use vague or short job descriptions
- Submit without reviewing the output first
- Forget to update contact information if needed

## Troubleshooting

**Problem:** Resume generation fails
- **Solution:** Check that your API key is set in `.env.local`
- **Solution:** Try uploading a different PDF format
- **Solution:** Paste your LinkedIn text instead of uploading

**Problem:** Output is not job-specific
- **Solution:** Paste a more detailed job description
- **Solution:** Include company name and industry in the job description

**Problem:** Missing information in resume
- **Solution:** Ensure your LinkedIn profile is complete
- **Solution:** Add missing details manually to the generated resume

## Need Help?

If you encounter issues, check:
1. Your API key is valid and in `.env.local`
2. Your LinkedIn PDF is not corrupted
3. The job description is clear and detailed
4. Your internet connection is stable
